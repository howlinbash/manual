Sway
====


## Cheatsheet


```bash
sup-shift-e                   # exit
sup-shift-q                   # close window
sup-q                         # lock

sup-e                         # default
sup-e                         # toggle between vertical and horizontal divide
sup-s                         # stacked
sup-w                         # tabbed
sup-shift-space               # floating windows

sup-d                         # dlauncher
sup-c                         # settings
sup-shift-c                   # reload config

sup-r                         # resize mode
with mouse: sup + right button drag border
with keys:  hjkl on focused window
```
